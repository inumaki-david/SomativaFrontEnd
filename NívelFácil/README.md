# Projeto Jogo Genius 🟢🔴🟡🔵 (nível fácil)

FEEDBACK:
O trabalho que escolhemos fazer para o nível facil foi o jogo de sequencia, escolhemos esse pois achamos o desafio divertido e interativo.
Para desenvolver o jogo, tivemos algumas dificuldades principalmente para realizar a parte da sequencia, pois não sabiamos como fazer as cores aparecerem e gravar a sequencia 
feita.
Usamos a inteligencia artificial para certas partes, mas com sua explicação conseguimos entender e fazer o que era preciso.
A parte que mais gostamos foi a do style, pois foi onde conseguimos ver o nosso jogo de verdade no site. Conseguimos mudar o estilo do jogo para o nosso gosto e
adicionar coisas a mais, como os sons.

FUNCIONALIDADES:
Primeiro, o código cria algumas variáveis para controlar o jogo. Uma guarda a sequência de cores gerada pelo jogo, outra guarda as cores que o jogador clica, além de uma variável para controlar o nível e outra para saber se o jogo está ativo. Também existe uma lista com as quatro cores usadas no jogo.
Depois, o código define sons para cada cor e um som de erro. Assim, sempre que uma cor aparece ou é clicada, um som é tocado.
Quando o jogador clica no botão Iniciar, o jogo reinicia todas as informações e começa a primeira rodada. A cada rodada, o jogo adiciona uma nova cor aleatória à sequência e mostra essa sequência piscando para o jogador.
Em seguida, o jogador precisa clicar nas cores na mesma ordem. Cada clique é registrado e comparado com a sequência do jogo. Se o jogador acertar tudo, ele passa para o próximo nível, onde a sequência fica maior. Se errar, o jogo termina, toca um som de erro e aparece uma mensagem perguntando se ele quer jogar novamente.
