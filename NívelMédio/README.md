# ⏳ Projeto Temporizador (nível médio)

Este é um projeto de um temporizador desenvolvido com html, css e js e o objetivo da aplicação é permitir que o utilizador defina um tempo em horas, minutos e segundos, e visualize a contagem decrescente em tempo real.

## 🚀 Funcionalidades

* **Entrada de Tempo Personalizada:** Campos específicos para horas, minutos e segundos com limites de validação.
* **Contagem Regressiva:** Sistema que subtrai o tempo segundo a segundo.
* **Controles de Fluxo:** * **Iniciar:** Começa a contagem com base nos valores inseridos.
    * **Pausar/Retomar:** Permite interromper o tempo e continuar do ponto onde parou.
    * **Reiniciar:** Para o temporizador e limpa todos os campos para uma nova utilização.
* **Interface Responsiva:** Design que se adapta a diferentes tamanhos de ecrã (computadores e telemóveis).
* **Alerta de Finalização:** Notificação visual quando o tempo chega a zero.

## 🧠 Relato de Desenvolvimento e Desafios

O desenvolvimento deste projeto foi uma jornada de aprendizagem prática muito importante:

* **HTML:** Esta etapa foi tranquila, pois já possuía conhecimentos prévios sobre a marcação e estruturação de elementos.
* **CSS:** A parte visual também foi acessível, embora tenha sido necessário consultar a **documentação oficial do CSS** para ajustar elementos específicos e implementar animações que eu ainda não conhecia.
* **JavaScript:** Foi o maior desafio. Embora a parte inicial (selecionar elementos pelo ID e criar funções básicas) tenha sido direta, a **lógica de programação** foi complexa. Pensar em como converter horas/minutos para segundos totais, gerir o estado de pausa e garantir que o intervalo parasse no momento certo exigiu bastante raciocínio. Contei com o auxílio de IA para estruturar melhor a lógica e as condições de verificação.
